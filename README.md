# IA Lab 🤖

> Um repositório centralizado para estudos e projetos em **Inteligência Artificial**

[![Status](https://img.shields.io/badge/Status-In%20Progress-yellow.svg)]()

## 📌 Sobre este repositório

Este é um espaço dedicado para organizar estudos, anotações e projetos de IA, com foco no **MBA em Engenharia de Software com IA** da Fullcycle. O objetivo é criar um recurso de aprendizado estruturado que também possa ajudar outras pessoas interessadas em IA.

## 🗂️ Estrutura do Repositório

```
ia-lab/
├── README.md              # Este arquivo
├── ROADMAP.md             # Progresso e planejamento
├── 📚 teoria/             # Fundamentais e conceitos teóricos
├── 📝 anotacoes/          # Anotações do MBA e estudos
├── 🏗️ projetos/           # Projetos completos
├── 🔬 experimentos/       # Experimentações e protótipos
├── 🛠️ utilitarios/        # Scripts, notebooks e ferramentas
└── 📖 recursos/           # Links, referências e datasets
```
